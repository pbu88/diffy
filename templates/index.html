Diffy - A tool for sharing diffs

Share your diffs and explain your ideas without committing

Mac:

  1. git diff | pbcopy and paste it on the textarea.
  2. Click "Diff me"

Linux:

  1. git diff > output.diff
  2. upload the output.diff file.

Windows:

  1. git diff > output.diff
  2. upload the output.diff file.

Note: It's not only restricted to git diff. Anything with a "diffy" output can be showed, like git show <commit number>
